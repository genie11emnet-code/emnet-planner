시작하기 — 팀원 이름 설정 방법
현재 표시된 이름(홍길동, 김지영 등)은 예시입니다. 아래 방법으로 실제 팀원 이름으로 교체하세요.
✅ 방법 1. UI에서 바로 변경 (권장)
  1. 1왼쪽 사이드바 하단 ⚙️ 팀원 · 캠페인 관리 클릭
  2. 2팀원 이름 입력칸에서 직접 수정
  3. 3색상 원 클릭으로 담당자 색상 변경
  4. 4저장 · 전체 반영 클릭 → 드롭다운·필터 모두 자동 업데이트
🔧 방법 2. HTML 파일 직접 수정
파일을 메모장 또는 VS Code로 열고 아래 부분을 찾아 수정하세요.
let MEMBERS = [ {name:'홍길동', color:'#4361EE'}, {name:'김지영', color:'#7C3AED'}, {name:'이민준', color:'#10B981'}, ... ];
💡 팁: 설정 변경 후 저장하면 사이드바 필터, 업무 등록 드롭다운, 담당자 표시가 모두 자동으로 반영됩니다.
📋
업무 등록하기
  1. 1우측 상단 ＋ 업무 등록 버튼 클릭
  2. 2업무명, 날짜, 담당자, 우선순위 입력
  3. 3카테고리 / 캠페인 / 마감일 선택 (선택사항)
  4. 4저장 클릭 → 목록에 즉시 반영
💡 캘린더 뷰에서 날짜 칸을 클릭하면 해당 날짜로 자동 입력됩니다.
✅
업무 완료 · 수정 · 삭제
  1. 1목록 왼쪽 체크박스 클릭 → 즉시 완료 처리 (취소선 표시)
  2. 2다시 클릭하면 완료 취소
  3. 3업무 행 전체 클릭 → 수정 팝업 열림
  4. 4행 오른쪽 🗑 버튼 → 삭제
💬
코멘트로 소통하기
  1. 1업무 행을 클릭 → 수정 팝업 열림
  2. 2팝업 하단 코멘트 입력창에 내용 입력
  3. 3전송 클릭 → 이름 + 시간 자동 기록
  4. 4파일을 공유하면 팀원이 함께 코멘트 확인 가능
💡 코멘트 작성자 이름은 사이드바 하단 카드에 설정된 이름으로 기록됩니다.

CSV 내보내기 · 구글 시트 연동
  1. 1상단 ⬇ CSV 내보내기 클릭
  2. 2현재 월 전체 업무가 파일로 다운로드
  3. 3구글 시트에서 가져오기 → 플래너 시트와 연동

⚙️
캠페인 목록 추가 · 변경
캠페인명은 설정 팝업에서 자유롭게 추가 / 수정 / 삭제할 수 있습니다.
  1. 1사이드바 하단 ⚙️ 팀원 · 캠페인 관리 클릭
  2. 2캠페인 목록 입력창에 캠페인명을 줄바꿈으로 구분해서 입력
  3. 3저장 · 전체 반영 클릭 → 사이드바와 업무 등록 드롭다운 자동 업데이트
💡 이 웹앱은 HTML 파일 하나로 동작합니다. 팀원과 공유하려면 파일을 전달하거나, 구글 드라이브에 업로드 후 링크를 공유하세요.

☁️ 팀 공유 설정
📌 팀 공유 방법 (무료 · 3분 설정)
  1. jsonbin.io 에 접속 → 회원가입 (구글 로그인 가능)
  2. 로그인 후 우측 상단 프로필 → API Keys 클릭
  3. Secret Key 복사 → 아래 API Key에 붙여넣기
  4. Bin ID는 비워두면 자동 생성됩니다
  5. 생성된 Bin ID를 팀원에게 공유하면 같은 데이터를 사용합니다
💡 데이터는 JSONBin.io 서버에 암호화되어 저장됩니다. 무료 플랜 기준 10,000회/월 요청, 100개 Bin, 각 최대 100KB 저장 가능합니다.
이미지가 많으면 용량 초과가 발생할 수 있습니다. 이미지가 많다면 URL 방식을 권장합니다.
